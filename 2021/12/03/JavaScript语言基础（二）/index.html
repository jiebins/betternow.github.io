JavaScript语言基础（二）——数据类型

发表于2021-12-03|更新于2022-01-03
|阅读量:

数据类型简介

ECMAScript共有6种简单数据类型，分别是：Undefined、Null、Boolean、Number、String和Symbol。还有一种复杂类型是Object。ECMAScript不能定义自己的数据类型，所有的值都可以用上述七种数据类型之一表示，并且一种数据类型可以当作多种数据类型来使用。

typeof操作符

typeof操作符用于确定变量的数据类型，返回值为字符串，含义如下表

返回值 含义
undefined 值未定义
boolean 布尔值
string 字符串
number 数值
object 对象或Null
function 函数
symbol 符号

Undefined类型

Undefined类型只有一个值undefined，使用let或var声明变量但没有初始化时相当于给变量赋予了undefined值。比如

由于上述特性的存在，所以在声明变量时没必要显式用undefined初始化变量。undefined在ES3之前并不存在，增加它的目的是为了明确空指针null和未声明变量的区别。

未声明变量和未初始化变量是有区别的，比如

let message;
console.log(age); // 直接报错
console.log(message); // 输出undefined

未声明的变量使用时会直接报错，但可以对它执行typeof操作，返回结果与Undefined类型的变量相同。

let message;
console.log(typeof age); // undefined
console.log(typeof message); // undefined

在声明变量时同时进行初始化，这样当typeof返回undefined时说明变量还未声明。

Null类型

Null类型也只有一个值null，表示一个空对象指针，因此使用typeof时会返回object。

对于将用来保存对象的变量，建议使用null来初始化，这样只要检查这个变量的值是否为null就可以知道这个变量是否在后来被重新赋予了一个对象的引用。

if (car != null) {
    // 说明car是一个对象的引用
}

注意

undefined是由null值派生而来，因此这两个值表面上是相等的，即

console.log(undefined == null); // true

使用==操作符比较null和undefined时始终返回true，但这并不说明这两个值是完全相等的，这个操作符的相关特性会在后面的文章中再详细说明。

Boolean类型

Boolean类型（布尔值类型）有两个字面值——true和false，两个字面值区分大小写，因此True和False是有效的标识符，而不是布尔值。通过Boolean()函数可以将其他类型的值转换为布尔值，转换规则如下

数据类型 转换为true的值 转换为false的值
Boolean true false
String 非空字符串 空字符串""
Number 非零数值（包括无穷） 0、NaN
Object 任意对象 null
Undefined 不存在 undefined

Number类型

Number类型使用IEEE 754格式表示整数和浮点数（双精度）。

整数

整数可以使用十进制、八进制和十六进制的字面量表示。十进制字面量直接写出来即可，比如

let intNum = 55;

八进制字面量第一个数字必须为0，接着是相应的八进制数字，如果字面量中包含的数字超出了应有的范围，后面的数字序列会被当成十进制数。

let octNum1 = 070; // 56
let octNum2 = 079; // 79
let octNum3 = 08; // 8

严格模式下前缀0的八进制字面量无效，会抛出语法错误，但可以使用0o作为八进制数的前缀。

十六进制字面量的前缀为0x，然后是十六进制数字，十六进制数字中的字母大小写均可，比如

let hexNum1 = 0xA; // 10
let hexNum2 = 0x1f; // 31

使用八进制和十六进制格式创建的数值在所有数学操作中都被视为十进制数值。

注意

JavaScript中存在+0和-0，两者在任何情况下都是相等的。

浮点数

浮点数必须包含小数点，而且小数点后必须至少有一个数字。小数点前不一定必须有整数，但推荐加上。由于存储浮点数使用的内存空间是整数的两倍，因此ECMAScript在以下情况下会把浮点数转换为整数

  • 小数点后面没有数字
  • 数值本身为整数，小数点后跟着0

数值范围

ECMAScript可以表示的数值范围有限，最小的数值保存在Number.MIN_VALUE，在大多数浏览器中这个值是5e-324；最大的数值保存在Number.MAX_VALUE中，在大多数浏览器中这个值是1.7976931348623157e+308。

如果计算的到的结果超出了这个范围，那么这个数值会被自动转换为特殊的无穷大Infinity值，超过最大值用Infinity表示，小于最小值用-Infinity表示。要确定一个值是否为Infinity值可以使用isFinite()函数，无穷则返回false，否则返回true，比如

数值转换

有三个函数用于将非数值转换为数值：Number()、parseInt()和parseFloat()。Number()是转型函数，可用于任何数据类型，另外两个函数主要用于将字符串转换为数值。Number()函数的转换规则如下

  • 布尔值，true转换为1，false转换为0。
  • 数值，直接返回。
  • null，返回0。
  • undefined，返回NaN
  • 字符串，应用以下规则
    • 如果字符串包含数值字符，包括数值字符前面带加减号的情况，则转换为一个十进制数值，忽略前面的零。
      • 如果字符串包含有效的浮点值格式如'1.1'，则会转换为相应的浮点值，忽略前面的零。
      • 如果字符串包含有效的十六进制格式如0xf，则会转换为与该十六进制值对应的十进制整数值。
      • 如果是空字符串则返回零。
      • 如果字符串包含除上述情况之外的其他字符，则返回NaN。
  • 对象，调用valueOf()方法，并按照上述规则转换返回的值。如果转换结果是NaN,则调用toString()方法，再按照转换字符串的规则转换。

在需要将字符串转换为数值时可以优先考虑parseInt()和parseFloat()两个函数。parseInt()函数可以接受一个字符串作为参数，将其转换为整数，字符串前面的空格会被忽略，从第一个非空格字符开始转换。如果第一个字符不是数值字符、加号或减号，parseInt()立即返回NaN，这一点与Number()不同，对于空字符串，Number()返回零，而parseInt()返回NaN。如果第一个字符是数值字符、加号或减号，则会继续检查每个字符直至字符串结尾或遇到非数值字符。

parseInt()也能识别不同的整数格式，比如八进制和十六进制。如果字符串以0x开头就会被解释为十六进制整数，如果以0开头并且紧跟着数值字符，在非严格模式下会被解释为八进制整数。

除了根据特定的前缀识别，parseInt()还可以接受第二个参数，用于指定底数。如果指定第二个参数，说明按指定参数的进制进行解析，对应的前缀可以省略。为避免解析出错，建议始终传给它第二个参数。以上内容的示例代码如下

parseFloat()函数与parseInt()类似，parseFloat()将一个字符串转换为一个浮点数。它也是从头到尾检测每个字符直到字符串末尾或解析到一个无效的浮点数值字符为止。这意味着小数点只能出现依次，后面出现的小数点无效，剩下的字符串也会被忽略。

parseFloat()始终忽略开头的零，只能识别十进制格式，十六进制的值始终返回零，并且不能指定底数。如果字符串表示整数（没有小数点或小数点后面只有一个零），则parseFloat()返回整数。

String类型

String字符串类型表示零或多个Unicode字符序列，字符串可以使用双引号(“)、单引号(‘)或反引号(`)表示。三种表示方法没有区别，但要注意引号需成对出现。

let str1 = 'Hello';
let str2 = "Hello";
let str3 = `Hello`;
let str4 = 'Hello"; // 出错，开头和结尾的引号不同

字符字面量

字符串类型包含一些字符字面量，用于表示非打印字符或有其他用途的字符。

字面量 含义
\n 换行
\t 制表
\b 退格
\r 回车
\f 换页
\\ 反斜杠（\）
\' 单引号（’）
\" 双引号（”）
\\` 反引号（`）
\xnn 以十六进制编码nn表示的字符，例如\x41等于A
\unnnn 以十六进制编码nnnn表示的Unicode字符，例如\u0313表示希腊字符∑

这些字符字面量作为单个字符被解释，因此在计算字符串长度时它们只算一个字符。字符串的长度可以通过其length属性获取。

console.log('sigma: \u03a3.'.length); // 9

字符串的特点

ECMAScript中的字符串是不可变的。
