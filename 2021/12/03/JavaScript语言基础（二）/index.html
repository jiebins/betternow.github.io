发表于2021-12-03|更新于2022-01-07

数据类型简介

ECMAScript共有6种简单数据类型，分别是：Undefined、Null、Boolean、Number、String和Symbol。还有一种复杂类型是Object。ECMAScript不能定义自己的数据类型，所有的值都可以用上述七种数据类型之一表示，并且一种数据类型可以当作多种数据类型来使用。

typeof操作符

typeof操作符用于确定变量的数据类型，返回值为字符串，含义如下表

返回值 含义
undefined 值未定义
boolean 布尔值
string 字符串
number 数值
object 对象或Null
function 函数
symbol 符号

下面是使用typeof操作符判断数据类型的例子

const obj = { a: 2 };
const fun = function () {}
const sym = Symbol();
console.log(typeof message);
console.log(typeof true);
console.log(typeof 'message');
console.log(typeof 56);
console.log(typeof obj);
console.log(typeof null);
console.log(typeof fun);
console.log(typeof sym);

Undefined类型

Undefined类型只有一个值undefined，使用let或var声明变量但没有初始化时相当于给变量赋予了undefined值。比如

由于上述特性的存在，所以在声明变量时没必要显式用undefined初始化变量。undefined在ES3之前并不存在，增加它的目的是为了明确空指针null和未声明变量的区别。

未声明变量和未初始化变量是有区别的，比如

let message;
console.log(age); // 直接报错
console.log(message); // 输出undefined

未声明的变量使用时会直接报错，但可以对它执行typeof操作，返回结果与Undefined类型的变量相同。

let message;
console.log(typeof age); // undefined
console.log(typeof message); // undefined

在声明变量时同时进行初始化，这样当typeof返回undefined时说明变量还未声明。

Null类型

Null类型也只有一个值null，表示一个空对象指针，因此使用typeof时会返回object。

对于将用来保存对象的变量，建议使用null来初始化，这样只要检查这个变量的值是否为null就可以知道这个变量是否在后来被重新赋予了一个对象的引用。

if (car != null) {
    // 说明car是一个对象的引用
}

注意

undefined是由null值派生而来，因此这两个值表面上是相等的，即

console.log(undefined == null); // true

使用==操作符比较null和undefined时始终返回true，但这并不说明这两个值是完全相等的，这个操作符的相关特性会在后面的文章中再详细说明。

Boolean类型

Boolean类型（布尔值类型）有两个字面值——true和false，两个字面值区分大小写，因此True和False是有效的标识符，而不是布尔值。通过Boolean()函数可以将其他类型的值转换为布尔值，转换规则如下

数据类型 转换为true的值 转换为false的值
Boolean true false
String 非空字符串 空字符串""
Number 非零数值（包括无穷） 0、NaN
Object 任意对象 null
Undefined 不存在 undefined

Number类型

Number类型使用IEEE 754格式表示整数和浮点数（双精度）。

整数

整数可以使用十进制、八进制和十六进制的字面量表示。十进制字面量直接写出来即可，比如

let intNum = 55;

八进制字面量第一个数字必须为0，接着是相应的八进制数字，如果字面量中包含的数字超出了应有的范围，后面的数字序列会被当成十进制数。

let octNum1 = 070; // 56
let octNum2 = 079; // 79
let octNum3 = 08; // 8

严格模式下前缀0的八进制字面量无效，会抛出语法错误，但可以使用0o作为八进制数的前缀。

十六进制字面量的前缀为0x，然后是十六进制数字，十六进制数字中的字母大小写均可，比如

let hexNum1 = 0xA; // 10
let hexNum2 = 0x1f; // 31

使用八进制和十六进制格式创建的数值在所有数学操作中都被视为十进制数值。

注意

JavaScript中存在+0和-0，两者在任何情况下都是相等的。

浮点数

浮点数必须包含小数点，而且小数点后必须至少有一个数字。小数点前不一定必须有整数，但推荐加上。由于存储浮点数使用的内存空间是整数的两倍，因此ECMAScript在以下情况下会把浮点数转换为整数

  • 小数点后面没有数字
  • 数值本身为整数，小数点后跟着0

数值范围

ECMAScript可以表示的数值范围有限，最小的数值保存在Number.MIN_VALUE，在大多数浏览器中这个值是5e-324；最大的数值保存在Number.MAX_VALUE中，在大多数浏览器中这个值是1.7976931348623157e+308。

如果计算的到的结果超出了这个范围，那么这个数值会被自动转换为特殊的无穷大Infinity值，超过最大值用Infinity表示，小于最小值用-Infinity表示。要确定一个值是否为Infinity值可以使用isFinite()函数，无穷则返回false，否则返回true，比如

数值转换

有三个函数用于将非数值转换为数值：Number()、parseInt()和parseFloat()。Number()是转型函数，可用于任何数据类型，另外两个函数主要用于将字符串转换为数值。Number()函数的转换规则如下

  • 布尔值，true转换为1，false转换为0。
  • 数值，直接返回。
  • null，返回0。
  • undefined，返回NaN
  • 字符串，应用以下规则
    • 如果字符串包含数值字符，包括数值字符前面带加减号的情况，则转换为一个十进制数值，忽略前面的零。
      • 如果字符串包含有效的浮点值格式如'1.1'，则会转换为相应的浮点值，忽略前面的零。
      • 如果字符串包含有效的十六进制格式如0xf，则会转换为与该十六进制值对应的十进制整数值。
      • 如果是空字符串则返回零。
      • 如果字符串包含除上述情况之外的其他字符，则返回NaN。
  • 对象，调用valueOf()方法，并按照上述规则转换返回的值。如果转换结果是NaN,则调用toString()方法，再按照转换字符串的规则转换。

在需要将字符串转换为数值时可以优先考虑parseInt()和parseFloat()两个函数。parseInt()函数可以接受一个字符串作为参数，将其转换为整数，字符串前面的空格会被忽略，从第一个非空格字符开始转换。如果第一个字符不是数值字符、加号或减号，parseInt()立即返回NaN，这一点与Number()不同，对于空字符串，Number()返回零，而parseInt()返回NaN。如果第一个字符是数值字符、加号或减号，则会继续检查每个字符直至字符串结尾或遇到非数值字符。

parseInt()也能识别不同的整数格式，比如八进制和十六进制。如果字符串以0x开头就会被解释为十六进制整数，如果以0开头并且紧跟着数值字符，在非严格模式下会被解释为八进制整数。

除了根据特定的前缀识别，parseInt()还可以接受第二个参数，用于指定底数。如果指定第二个参数，说明按指定参数的进制进行解析，对应的前缀可以省略。为避免解析出错，建议始终传给它第二个参数。以上内容的示例代码如下

parseFloat()函数与parseInt()类似，parseFloat()将一个字符串转换为一个浮点数。它也是从头到尾检测每个字符直到字符串末尾或解析到一个无效的浮点数值字符为止。这意味着小数点只能出现依次，后面出现的小数点无效，剩下的字符串也会被忽略。

String类型

String字符串类型表示零或多个Unicode字符序列，字符串可以使用双引号(“)、单引号(‘)或反引号(`)表示。三种表示方法没有区别，但要注意引号需成对出现。

let str1 = 'Hello';
let str2 = "Hello";
let str3 = `Hello`;
let str4 = 'Hello"; // 出错，开头和结尾的引号不同

字符字面量

字符串类型包含一些字符字面量，用于表示非打印字符或有其他用途的字符。

字面量 含义
\n 换行
\t 制表
\b 退格
\r 回车
\f 换页
\\ 反斜杠（\）
\' 单引号（’）
\" 双引号（”）
\\` 反引号（`）
\xnn 以十六进制编码nn表示的字符，例如\x41等于A
\unnnn 以十六进制编码nnnn表示的Unicode字符，例如\u0313表示希腊字符∑

这些字符字面量作为单个字符被解释，因此在计算字符串长度时它们只算一个字符。字符串的长度可以通过其length属性获取。

console.log('sigma: \u03a3.'.length); // 9

字符串的特点

ECMAScript中字符串是不可变的。这就意味着字符串一旦创建它的值就不能被修改，比如

尽管修改了str的首位，但可以看到输出的结果仍为Javascript，这便说明字符串创建后我们无法对它的字符进行修改，即不可变。另外，这还说明字符串的存储方式类似于数组，我们可以使用访问数组的方式来访问字符串。

虽然不能直接修改字符串中的字符，但是我们可以直接修改整个字符串，比如

str = 'KavaScript';

这样就能达到我们前面想要的效果了。在这个过程中，后台先分配空间给字符串KavaScript，然后销毁了原来的字符串JavaScript，最后把KavaScript赋给变量str。

下面一个例子说明这种不可变性会导致拼接字符串的效率低下。

let lang = 'Java';
lang = lang + 'Script';

在拼接'Java'和'Script'的过程中，首先会分配10个字符的空间存储字符串'JavaScript'，然后销毁'Java'和'Script'，最后把'JavaScript'分配给变量lang。每次拼接都要新建一块空间来存储拼接后的字符串，还要销毁原来的字符串，这就降低了运行效率。

转换为字符串

有两种方式将一个值转换为字符串——toString()方法和String()函数。几乎所有值都有toString()方法，这个方法返回当前值的等价字符串，比如

let age = 11;
let ageAsString = age.toString(); // 字符串'11'
let found = true;
let foundAsString = found.toString(); // 字符串'true'

toString()方法可用于数值、布尔值、对象和字符串值（返回自身的副本）。null和undefined没有toString()方法。多数情况下toString()方法不接收任何参数，但在对数值调用这个方法时可以传入一个底数参数，即按什么进制来输出数值的字符串表示，默认不传参数的情况下返回十进制。

如果不确定一个值是否为null或undefined，可以使用String()转型函数。String()函数遵循以下规则

  • 如果值有toString()方法则调用该方法（不传参数）并返回结果。
  • 如果值是null则返回'null'。
  • 如果值是undefined则返回'undefined'。

模板字面量

ES6新增了使用模板字面量定义字符串的能力，使用反引号定义模板字面量，模板字面量可以保留字符串原来的格式，比如

let myMultiLineString = 'first line \nsecond line';
// 直接在first line后面换行
let myMultiLineTemplateLiteral = `first line
second line`;
console.log(myMultiLineString);
// first line
// second line
console.log(myMultiLineTemplateLiteral);
// first line
// second line
console.log(myMultiLineString == myMultiLineTemplateLiteral); // true

模板字面量会保持反引号内部的空格，某些看起来相同的字符串实际上是两个长度不同的字符串，比如

let temp1 = `hello
world`;
let temp2 = `hello
             world`;
let temp3 = `
hello
world`
console.log(temp1.length); // 11
console.log(temp2.length); // 24
console.log(temp3[0] == '\n'); // true

temp2的world前还有13个空格，因此长度为24，而temp1中没有空格。temp3前面直接换行，所以第一个是换行符。

字符串插值

模板字面量最常用到特性就是字符串插值，也就是在字符串的某个位置插入一个变量。这样看来，模板字面量不是字符串，而是一种特殊的句法表达式，只是求值的结果是字符串而已。模板字面量在定义时立即求值并转换为字符串实例。只需把${}放在模板字符串中需要插入值的地方并在花括号内填入相应的变量名即可完成字符串插值，比如

let value = 5;
let exponent = 'second';
let interpolatedTemplateLiteral = `${ value } to the ${ exponent } power is ${value * value}`;
// 字符串插值实际上是字符串的拼接，与下面的语句等价
let interpolatedString = value + ' to the ' + exponent + ' power is ' + (value * value);
console.log(interpolatedTemplateLiteral); // 5 to the second power is 25
console.log(interpolatedString); // 5 to the second power is 25

所有插入的值和表达式都会使用toString()方法强制转换为字符串，任何表达式都可以用于插值。插入的模板字面量的反引号无需转义，还可以插入自己之前的值，比如

let foo = { toString: () => 'World' }
console.log(`Hello, ${ `World` }!`); // Hello, World!
// 调用toString方法
console.log(`Hello, ${ foo }!`); // Hello, World!
// 插入自己之前的值
let value = '';
value = `${ value }abc`;
console.log(value); // abc
value = `${ value }abc`;
console.log(value); // abcabc

模板字面量标签函数

标签函数用于自定义插值行为，标签函数会接收被插值记号${}分隔后的模板和对每个表达式求值的结果，返回值是对模板字面量求值得到的字符串，比如

let a = 6;
let b = 9;
function simpleTag(strings, aVal, bVal, sum) {
  console.log(strings);
  console.log(aVal);
  console.log(bVal);
  console.log(sum);
  return 'foobar';
}
let untaggedResult = `${ a } + ${ b } = ${ a + b }`;
let taggedResult = simpleTag`${ a } + ${ b } = ${ a + b }`; // 调用不用括号
// ['', ' + ', ' = ', '']
// 6
// 9
// 15
console.log(untaggedResult); // '6 + 9 = 15'
console.log(taggedResult); // 'foobar'

可以看到第一个参数是还没插值时的字符串数组，数组的长度为插值的个数加一，系统以插值的部分为分割点，将模板字符串模板字面量不参与插值的部分分隔成了若干个部分。第二个参数开始就是待插值的表达式的计算结果，返回值可以自定义，可以返回其他字符串。由于待插值的表达式的数量是可变的，所以可以使用剩余操作符（…）在函数参数部分将它们收集到数组中

function simpleTag(strings, ...expressions) {
  console.log(strings);
  for (const expression of expressions) {
      console.log(expression);
  }
  return 'foobar';
}
console.log(simpleTag`${ 6 } + ${ 9 } = ${ 6 + 9 }`);
// ['', ' + ', ' = ', '']
// 6
// 9
// 15
// 'foobar'

如果想返回拼接后的结果（也就是模板字面量不调用标签函数时返回的结果），可以这么写

let a = 6;
let b = 9;
function zipTag(strings, ...expressions) {
  return strings[0] + expressions.map((e, i) => `${e}${strings[i + 1]}`).join('');
}
console.log(`${ a } + ${ b } = ${ a + b }`); // '6 + 9 = 15'
console.log(zipTag`${ a } + ${ b } = ${ a + b }`); // '6 + 9 = 15'

原始字符串

对于会转义的字符，比如换行符和Unicode字符，可以使用标签函数String.raw来显示原始的内容，比如

console.log(`\u00A9`); // ©
console.log(String.raw`\u00A9`); // \u00A9

对于不是转义的换行符这个方法是不会得到原始的字符，比如

console.log(String.raw`first
seconde`);
// first
// second

也可以利用标签函数第一个参数的raw属性取得每个字符串的原始内容

function printRaw(strings) {
    console.log(strings.raw);
}
printRaw`\u00A9${ 'and' }\n`; // ['\u00A9', '\n']

Symbol类型

Symbol（符号）类型是ES6新增的数据类型，主要用于保证对象的属性是唯一的，避免发生属性冲突。当一个对象有着不胜枚举的属性时，如果你想给它添加一个新的属性，那么你就要考虑该对象是否已经存在你要添加的属性，此时就要在对象的所有属性中进行查找，使用符号作为属性就避免查找这个操作，因为符号定义的属性是唯一的，不必担心与前面的属性冲突。

符号的基本用法

符号使用Symbol()函数初始化，由于它也是原始类型，所以typeof对符号返回symbol。

let sym = Symbol();
console.log(typeof sym); // symbol

任何两个符号都是不一样的。定义符号时可以传入一个字符串参数，作为该符号的描述，然而即使两个符号的描述相同，它们也是不相等的。

let sym1 = Symbol();
console.log(sym1); // Symbol()
let sym2 = Symbol();
console.log(sym1 === sym2); // false
let sym3 = Symbol('foo');
console.log(sym3); // Symbol(foo)
let sym4 = Symbol('foo');
console.log(sym3 === sym4); // false

符号没有字面量语法，只需创建Symbol()实例用作对象的新属性就可以保证它不会覆盖已有属性。此外，Symbol()函数不能与new关键字一起作为构造函数使用，避免创建符号包装对象，如果确实需要使用符号包装对象，可以借用Object()函数。

let mySymbol = new Symbol(); // TypeError: Symbol is not a constructor
let mySymbol = Symbol();
let mySymbol = Object(mySymbol);

全局符号注册表

如果需要共享和重用符号实例，可以使用一个字符串作为键，在全局符号注册表中创建并重用符号。使用Symbol.for()方法可以完成上述操作。

let fooGlobalSymbol = Symbol.for('foo');
console.log(typeof fooGlobalSymbol); // symbol

第一次使用某个字符串调用Symbol.for()时，它会检查全局符号注册表，如果发现不存在对应的符号就会生成一个新的符号实例并添加到注册表中，如果发现存在对应的符号就返回该符号实例。

let fooGlobalSymbol = Symbol.for('foo');
let otherGlobalSymbol = Symbol.for('foo');
console.log(fooGlobalSymbol === otherGlobalSymbol); // true

使用Symbol()定义的符号是不会写入全局符号注册表中的，并且采用相同的符号描述，在全局符号注册表中定义的符号与Symbol()定义的符号不同

let localSymbol = Symbol('foo');
let globalSymbol = Symbol.for('foo');
console.log(localSymbol === globalSymbol); // false

全局符号必须使用字符串键来创建，因此作为参数传给Symbol.for()的任何值都会被转换为字符串，并且注册表中使用的键会被用作符号描述

let emptyGlobalSymbol = Symbol.for();
console.log(emptyGlobalSymbol); // Symbol(undefined)

可以使用Symbol.keyFor()来查询全局注册表，这个方法接收符号，返回该符号的字符串键，如果不是全局符号，则返回undefined，如果不是符号，则抛出错误。

let s = Symbol.for('foo');
let s2 = Symbol('bar');
console.log(Symbol.keyFor(s)); // foo
console.log(Symbol.keyFor(s2)); // undefined
console.log(Symbol.keyFor(123)); // TypeError: 123 is not a Symbol.

使用符号作为属性

使用符号作为属性有以下四种方法

let s1 = Symbol('foo');
    s2 = Symbol('bar');
    s3 = Symbol('baz');
    s4 = Symbol('qux');
    s5 = Symbol('qqq');
// 第一种方法，使用对象字面量
let o = {
    [s1]: 'foo val'
};
// 第二种方法，使用[]
o[s2] = 'bar val';
// 第三种方法，使用Object.defineProperty()
Object.defineProperty(o, s3, { value: 'baz val' });
// 第四种方法，使用Object.defineProperties()
Object.defineProperties(o, {
    [s4]: 'qux val',
    [s5]: 'qqq val'
})
console.log(o);
// {Symbol(foo): 'foo val', Symbol(bar): 'bar val', Symbol(baz): 'baz val', Symbol(qux): 'qux val', Symbol(qqq): 'qqq val'}

  • 注意

    • 第一种方法使用对象字面量时只能在计算属性语法中使用符号作为属性。计算属性使用方括号[]在对象字面量中定义，可以将一个表达式定义为对象的属性，属性名为表达式1的字符串结果

      let first = ‘second’;
      let o = {

            [first]: 2
        };
        console.log(o); // {second: 2}

使用Object.getOwnPropertyNames()可以返回对象实例的字符串属性数组，但不能返回符号属性，而使用Object.getOwnPropertySymbols()可以返回对象实例的符号属性数组，不能返回字符串属性，Reflect.ownKeys()可以返回两种属性。

let s1 = Symbol('foo');
let s2 = Symbol('bar');
let o = {
    [s1]: 'foo val',
    [s2]: 'bar val',
    baz: 'baz val',
    quz: 'qux val'
}
console.log(Object.getOwnPropertyNames(o)); // ["baz", "qux"]
console.log(Object.getOwnPropertySymbols(o)); // [Symbol(foo), Symbol(bar)]
console.log(Reflect.ownKeys(o)); // ["baz", "qux", Symbol(foo), Symbol(bar)]

如果不先定义符号再将其作为对象的属性，而是在对象中直接使用Symbol()函数定义的符号作为属性，那么必须遍历所有符号属性才能找到相应的属性键，因为符号属性是对内存中的符号的一个引用，不预先定义符号，就访问不到定义好的属性，在查找对象对应属性时无法使用符号直接获取对应的属性值。

let o = {
    [Symbol('foo')]: 'foo val',
    [Symbol('bar')]: 'bar val'
}
let barSymbol = Object.getOwnPropertySymbols(o).find((symbol) => symbol.toString().match(/bar/));
console.log(barSymbol);

Object类型

对象是一组数据和功能的集合，通过new操作符后跟对象类型的名称来创建。可以通过创建Object实例来创建对象，然后再给对象添加属性和方法，比如

let o = new Object();
// 括号可以省略，但是不推荐
let o1 = new Object;

Object是所有对象的基类，任何对象都会有Object对象的方法和属性。Object对象实例有以下的属性和方法：

  • constructor：用于创建当前对象的函数，类似于C++中的构造函数。
  • hasOwnProperty(propertyName)：用于判断当前对象实例上是否存在给定的属性，propertyName为属性名，必须是字符串或符号。
  • isPrototypeOf(object)：用于判断当前对象是否为另一个对象的原型（后面会介绍原型）。
  • propertyIsEnumerable(propertyName)：用于判断给定属性是否可以使用for-in语句枚举。属性名propertyName必须是字符串。
  • toLocaleString()：返回对象的字符串表示，该字符串反映对象所在的本地化执行环境。
  • toString()：返回对象的字符串表示。
  • valueOf()：返回对象对应的字符串、数值或布尔值表示，通常与toString()的返回值相同。
