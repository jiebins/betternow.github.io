    HTML基础语法

    发表于 2021-11-30

    1 基本介绍

    HTML（HyperText Markup Language，超文本标记语言）是一种告诉浏览器如何组织页面的标记语言，由一系列元素（elements）组成。

    超文本的含义

    • 可以加入图片、声音、动画、多媒体等内容。（超越文本限制）
    • 可以从一个文件跳转到另一个文件，与世界各地主机的文件连接。（超级链接文本）

      HTML作用

      网页由网页元素组成，这些元素是利用HTML标签描述出来，然后通过浏览器解析就可以显示给用户

    2 HTML元素

    2.1 HTML元素结构

    HTML元素示例

    <p>This is a html element.</p>

    该元素主要部分有

    1. 开始标签。<p>为开始标签，p为元素名称，表示元素从这里开始。
    2. 结束标签。</p>为结束标签，在元素名之前多了一个/，表示元素到这里结束。
    3. 内容。<p>与</p>之间的内容为元素内容。
    4. 元素。开始标签+内容+结束标签构成一个元素。

    有些元素不包含内容和结束标签，只有一个标签，通常用来在此元素位置插入或嵌入一些东西，例如

    <img alt="img" src="https://hanyu.baidu.com/shici/detail?pid=b3c0f7fa848f494aa9e4868aa5bbcd9b&from=kg0&highlight=%E9%87%91%E6%A8%BD%E6%B8%85%E9%85%92%E6%96%97%E5%8D%81%E5%8D%83">

    2.2 元素嵌套

    可以把元素嵌套在另一个元素中，比如

    <p>This is a html <b>element</b>.</p>

    嵌套时要注意结束标签的顺序，先开始的后结束，后开始的先结束。

    2.3 元素属性

    给元素添加一个class属性，格式如下

    <p class="paragraph">This is a html element.</p>

    属性的具体格式如下

    <开始标签+空格+属性名称+等号=+属性值>(</结束标签>)

    如果有多个属性则每个属性与前一个属性之间用空格隔开，属性值用双引号””括起来。

    属性注意点

    布尔属性

    没有值的属性也是合法的，这些属性被称为布尔属性，只能有与属性名一样的属性值，例如disabled属性。

    <input type="text" disabled="disabled">

    可以写成

    <input type="text" disabled>

    省略引号的属性值

    不给属性值加引号在某种情况下时允许的，比如

    <a href=https://www.baidu.com>百度一下</a>

    但如果属性值中有空格浏览器会将其识别为若干个属性，比如
    <a href=www.baidu.com title=baidu yixia>百度一下</a>
    title属性的值时baidu，yixia变成了布尔属性。
    ####正确使用引号
    属性可以用单引号或双引号括起来，但单双引号在一个属性值里不能混合使用，即不能单引号开始双引号结束或者双引号开始单引号结束。

    可以在属性中嵌套使用引号，比如
    <a href="https://www.baidu.com" title="baidu 'yi'xia">百度一下</a>

    3 HTML骨架

    HTML文档的基本骨架为

    <!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>title</title>
    </head>
    <body>body</body>
</html>

    其中<!DOCTYPE>在文档最前面声明，处于HTML标签之前，告诉浏览器文档使用哪种HTML或XHTML规范解析文档。表示html5规范；html元素为根元素，包含整个页面的所有元素；head元素保存页面的一些元数据；body元素包含页面的主体内容。

    4 HTML中的特殊字符

    空白符

    在HTML元素中无论使用多少空格，当渲染这些代码时，解释器会将连续出现的空白字符减少为一个单独的空白符。

    其他特殊字符

    HTML中有些字符是HTML语法的一部分，不方便直接> > 使用，可以使用字符引用来表示字符的特殊编码。

    原义字符 等价字符引用
    < &lt;
    > &gt;
    “ &quot;
    ‘ &apos;
    & &amp;
    特殊字符以&开头，以分号;结尾。

    5 HTML注释

    和其他编程语言一样，HTML也有一种机制来书写注释，注释会被浏览器忽略，用来给代码做标记或解释。注释用<!--和-->包括起来，比如

    <!--这是一个p标签-->

    <p>this is a label.</p>
