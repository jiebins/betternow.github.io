常用HTML标签及其属性介绍

发表于 2021-12-01 更新于 2021-12-02

1 head标签介绍

head标签里的内容是不会显示在页面上的，它的作用是保存页面的一些元数据。

2 head中常用元素

2.1 title标签

title标签可以为HTML文档添加标题，标题会显示在浏览器的标签栏，比如

<title>百度一下</title>

效果如下

2.2 meta标签

meta标签用于为HTML文档添加元数据（描述数据的数据），以下是一些常用的方法。

2.2.1 指定字符编码

通过meta标签的charset属性可以指定文档的编码类型，比如

<meta charset="utf-8">

以上代码指定了网页以utf-8编码来显示字符。常用的字符集还有

字符集名称 解释
GB2312 6763个汉字和682个非汉字图形字符
BIG5 繁体中文
GBK 全部中文字符
UTF-8 全世界所有国家需要用到的字符

2.2.2 添加作者和描述

meta标签的name属性用于指定meta元素的类型，说明meta元素包含什么类型的信息，content属性指定实际的元数据内容。利用这两个属性可以给HTML文档添加作者和文档的简要描述，比如

<meta name="author" content="海里睡人">

<meta name="description" content="一些描述">

利用description指定包含关于页面内容关键字的描述可以让页面在搜索引擎的相关搜索中出现更多。

2.3 link标签

link标签主要用于定义文档与外部资源的关系，使用rel属性表示当前文档与被link文档之间的关系，href属性表示被link文档的位置，type属性表示被link文档的MIME类型。常见用法有

2.3.1 添加自定义图标

前面说到可以利用title标签给HTML文档添加标题，使用link标签可以在标题前添加自定义图标，比如

<link rel="shortcut icon" href="favicon.png" type="image/png">

2.3.2 应用CSS文件

使用link标签可以引入外部CSS文件，比如

<link rel="stylesheet" href="cssfile.css">

3 body标签介绍

body标签用于装载页面的主体内容，比如文本、链接、图像、表格等，下面分类介绍一些body中常用的标签。

排版标签主要和CSS搭配使用，显示网页结构的标签，是网页布局最常用的标签。

3.1.5 div和span标签

div和span标签是使用最多的布局标签。div标签<div></div>可以把文档分割成独立、不同的部分，并且本身没有任何格式。span标签<span></span>与div标签基本相同，但span标签用在行内。div标签一行只能放一个，span标签一行能放多个。效果如下

3.2 文本格式化标签

文本格式化标签用于给文本加修饰，比如加粗、斜体、删除线和下划线，效果如下

标签 显示效果
<b>加粗</b>或<strong>加粗</strong> 加粗或加粗
<i>斜体</i>或<em>斜体</em> 斜体或斜体
<s>删除线</s>或<del>删除线</del> 删除线或删除线
<u>下划线</u>或<ins>下划线</ins> 下划线或下划线

3.3 图像标签

图像标签<img src="image-url">用于显示图片，src属性用于指定图片的路径和文件名，该属性不能省略。图像标签常用的属性还有

属性 值 含义
src url 图片路径
alt 文本 图片不能显示时的替换文本
title 文本 鼠标悬停时显示的内容
width 像素 设置图像宽度
height 像素 设置图像高度（设置宽高任一个另一个等比例缩放）
border 数字 设置图像边框宽度

alt与title的区别

  • alt是img标签的特有属性，是图片内容的等价描述，可提高图片的可访问性，搜索引擎会分析。
  • title是鼠标悬停在图片上显示的内容。

找个网图举栗子

<img src="https://img0.baidu.com/it/u=1582506239,925147619&fm=26&fmt=auto" alt="袁绍" title="袁绍" width=300>

<img src="caozhang" alt="曹彰" title="曹彰" width=300 border=1>

3.4 链接标签

标签<a>定义超链接，用于从一个页面链接到另一个页面，常用属性有

属性 含义
href 必须属性，指定链接目标的url，当为标签应用href属性时它就具有超链接功能
target 用于指定链接页面的打开方式，取值有_blank、_self等，其中_blank表示在新标签页打开链接，默认为_self
title 鼠标悬停在链接上显示的内容

外部链接需添加http或https，否则会报错，比如

<a href="www.baidu.com">百度一下</a>

百度一下

<a href="https://www.baidu.com">百度一下</a>
