1、HTML基本语法

1.1 基本介绍

HTML（HyperText Markup Language，超文本标记语言）是一种告诉浏览器如何组织页面的标记语言，由一系列元素（elements）组成。

超文本的含义

  • 可以加入图片、声音、动画、多媒体等内容。（超越文本限制）
  • 可以从一个文件跳转到另一个文件，与世界各地主机的文件连接。（超级链接文本）

    HTML作用

    网页由网页元素组成，这些元素是利用HTML标签描述出来，然后通过浏览器解析就可以显示给用户

1.2 HTML元素

1.2.1 HTML元素结构

HTML元素示例

<p>This is a html element.</p>

该元素主要部分有

  1. 开始标签。<p>为开始标签，p为元素名称，表示元素从这里开始。
  2. 结束标签。</p>为结束标签，在元素名之前多了一个/，表示元素到这里结束。
  3. 内容。<p>与</p>之间的内容为元素内容。
  4. 元素。开始标签+内容+结束标签构成一个元素。

有些元素不包含内容和结束标签，只有一个标签，通常用来在此元素位置插入或嵌入一些东西，例如

<img alt="img" src="https://hanyu.baidu.com/shici/detail?pid=b3c0f7fa848f494aa9e4868aa5bbcd9b&from=kg0&highlight=%E9%87%91%E6%A8%BD%E6%B8%85%E9%85%92%E6%96%97%E5%8D%81%E5%8D%83">

1.2.2 元素嵌套

可以把元素嵌套在另一个元素中，比如

<p>This is a html <b>element</b>.</p>

嵌套时要注意结束标签的顺序，先开始的后结束，后开始的先结束。

1.2.3 HTML骨架

HTML文档的基本骨架为

<html>
    <head></head>
    <body></body>
</html>

其中html元素为根元素，包含整个页面的所有元素，head元素保存页面的一些元数据，body元素包含页面的主体内容。
